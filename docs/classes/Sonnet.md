# Sonnet Class Documentation

## Methods

### startPritheeGame
This is the method that starts this whole game and is the main method 
inside the Sonnet Class, it calls other methods mentioned later on.


### printSonnetString
This is the method that prints the Sonnet string up to a specified index at
which then it will stop and then print out underscores as needed.

### getGuess
This is the method that will take in a correct string input and scanner and 
then take in input from the user via scanner and then normalize the input
and then output a boolean value based on if it was a correct guess
or not.
